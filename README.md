# calc2026 — Высокоточный калькулятор на QML и C++

Высокоточный калькулятор с поддержкой 25 знаков после запятой, 
кастомным интерфейсом на QML и скрытым меню. 
Создан как тестовое задание с использованием MSVC 2019 и Qt 6.5.2.


## 📁 Структура проекта

```plaintext
calculator2026/
├── CMakeLists.txt # Основной файл сборки (CMake)
├── CMakeLists.txt.user # Настройки Qt Creator
├── main.cpp # Точка входа: инициализация Qt + загрузка QML
├── calculator.h / calculator.cpp # Ядро: классы BigNumber и Calculator
├── main.qml # Главный интерфейс (окно калькулятора)
├── SecretMenu.qml # Скрытое меню (активируется по комбинации)
├── SecretCodeLogic.qml # Логика активации скрытых функций
├── CustomButton.qml # Кастомные кнопки с анимацией
├── mainwindow.ui # (Опционально) UI-дизайн для ручной настройки
├── OpenSans.ttf # Шрифт по умолчанию (загружается через QML)
├── resources.qrc # Ресурсы: шрифт, упакованные .qml-и
├── release/ # Собранный исполняемый файл (appcalc2026.exe)
└── README.md


## 💡 Функционал
Функция / Описание
Высокая точность / Класс BigNumber поддерживает до 25 знаков после запятой — без потерь при сложении/умножении и т.д.
Кастомный интерфейс / Все кнопки — CustomButton.qml, анимации, стиль OpenSans.ttf
Скрытое меню / Активируется комбинацией: удержанием кнопки = от 4-х секунд и вводом 123 — открывает SecretMenu.qml
Логика на C++ / Все вычисления — в Calculator (QObject), безопасно интегрирован с QML через setContextProperty()
Ресурсы / Шрифт OpenSans.ttf загружается через resources.qrc — не требует установки в систему


## 🧠 Ключевые технологии
C++: BigNumber — класс с ручной арифметикой (строки, не double!), Calculator — QObject с сигналами/слотами
QML: Динамический UI, реактивные свойства, анимации, интеграция с C++
CMake: Современная сборка (не .pro)
Qt Quick Controls 2: Современные элементы управления
Ресурсы: Шрифт встроен в бинарник — никаких зависимостей

✅ Не double — а точные вычисления — исключение проблем с плавающей запятой
✅ Скрытые функции — креатив и внимание к деталям
✅ CMake + QML + C++ — современный стек, как в реальных проектах
✅ Шрифт в ресурсах — подходит для кроссплатформенности
✅ Чистый код, структурированная архитектура — легко читать и поддерживать
